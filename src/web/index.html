Bem-vindo ao Alfred!

  • alfred toque [música] — Toca músicas do YouTube (Lavalink)
  • Comando de Voz — Entre no canal e fale "Alfred, toque [música]"
  • alfred playlist salvar [nome] — Salva fila atual como playlist
  • alfred lembre que [fato] — Salva memórias de longo prazo
  • Pergunte qualquer coisa — IA com Llama 4 Maverick

Dicas de Uso

  • 🎤 Voz: Entre num canal de voz e fale com Alfred para tocar músicas!
  • 🎵 Música: Use pula, pausa, para para controlar
  • 📀 Playlist: alfred playlist listar para ver suas playlists salvas
  • 🧠 Memória: Alfred lembra o que você ensina entre conversas
  • 🌍 Traduzir: alfred traduza para inglês: Olá mundo
  • ⛅ Clima: alfred como está o tempo em São Paulo?
